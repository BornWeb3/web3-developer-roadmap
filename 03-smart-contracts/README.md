# День 3: как вызвать смарт контракт и что такое ABI

## Что такое смарт контракт

Смарт контракт это программа, которая лежит в блокчейне и выполняется по правилам которые в ней написаны.  

Ты не можешь просто взять и изменить её код после деплоя. Ты можешь только вызывать её функции.  

Пример  

Есть контракт токена  
У него есть функция transfer  
Когда ты вызываешь её, контракт переводит токены  

## Как вообще происходит вызов

Ты не обращаешься к контракту напрямую  

Ты создаёшь транзакцию  
в которой указываешь  

адрес контракта  
какую функцию вызвать  
какие параметры передать  

После этого сеть выполняет код контракта  

Пример  

Ты хочешь перевести токены  
в транзакции указываешь функцию transfer  
адрес получателя  
и количество  

Контракт получает эти данные и выполняет логику  

## Что такое ABI

ABI это описание контракта  

Это как инструкция которая говорит  

какие функции есть  
какие параметры они принимают  
что они возвращают  

Без ABI ты не сможешь правильно взаимодействовать с контрактом  

Простой пример  

Представь что контракт это пульт  
а ABI это инструкция к нему  

Без инструкции ты не знаешь какую кнопку нажать  

## Как это выглядит логически

Ты пишешь код  
код использует ABI  
формирует вызов функции  
отправляет транзакцию  

контракт получает данные  
выполняет функцию  
возвращает результат  

## Виды вызовов

Есть два типа взаимодействия  

чтение данных  
и запись данных  

Чтение  

ты просто получаешь данные  
транзакция не создаётся  
газ не тратится  

Пример  

узнать баланс токена  

Запись  

ты меняешь состояние  
создаётся транзакция  
нужно платить газ  

Пример  

перевод токенов  

## Простой пример из жизни

Контракт это банкомат  

ABI это инструкция к банкомату  

ты выбираешь действие  
вводишь данные  
банкомат выполняет операцию  

## Главная мысль

Ты не работаешь с контрактом напрямую  

Ты работаешь через ABI  

ABI говорит как с ним общаться  
а транзакции позволяют выполнять действия  

Без этого невозможно взаимодействие со смарт контрактами
